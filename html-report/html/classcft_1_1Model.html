Detailed Description

Class that is abstraction of a 3D-Model.

This class stores the vertices of the model along with the color values. It also exposes variables to translate and rotate the object. The matrix stores different Matrices like projection and transform. It has functions to load and store the 3D-Model in a file. Other functions to add and remove triangles from the model
